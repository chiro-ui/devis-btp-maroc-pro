# Vision

Notre mission est de créer la plateforme BTP la plus complète du Maroc.

## Valeurs

- Qualité
- Simplicité
- Sécurité
- Performance
- Gratuité pour la version communautaire
- Évolutivité

## Vision

Une seule plateforme regroupant :

- Devis
- Métré
- DQE
- BPU
- IA
- BIM
- Formation

Développée pour le marché marocain.
