# Architecture de Devis BTP Maroc Pro

## Objectif

Développer une plateforme SaaS moderne pour la gestion des devis, métrés et projets BTP au Maroc.

## Frontend

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui

## Backend

- Supabase
- PostgreSQL
- Auth
- Storage
- Edge Functions

## Déploiement

- Vercel

## Applications

- Web
- Android
- Windows (plus tard)

## Langues

- Français
- العربية

## Devise

- Dirham marocain (MAD)

## Modules

- Authentification
- Clients
- Projets
- BPU
- DQE
- Métré
- Devis
- Facturation
- Rapports
- Assistant IA
