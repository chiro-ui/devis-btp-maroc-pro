# 🏗️ Devis BTP Maroc Pro

Le premier écosystème marocain de gestion des devis, métrés, BPU et IA pour le secteur du BTP.

## 🎯 Objectif

Créer une plateforme moderne destinée aux :

- Ingénieurs
- Architectes
- Métreurs
- Entreprises de construction
- Étudiants
- Maîtres d'ouvrage

## Modules

- 📐 Métré
- 📑 DQE
- 📚 BPU Maroc
- 💰 Devis estimatif
- 📊 Analyses de prix
- 🧱 Matériaux
- 👷 Main-d'œuvre
- 🚜 Matériel
- 📄 Export PDF / DOCX / Excel
- 🤖 FABY AI

## Technologies

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- PostgreSQL
- Vercel
- Flutter
- Python
